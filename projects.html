Jarvis assisstance

Description:-

Jarvis is a Python-based virtual assistant leveraging libraries like OpenCV for computer vision, Mediapipe for hand and face tracking, and Pyttsx3 for text-to-speech conversion. It uses SpeechRecognition for voice input and Natural Language Toolkit (NLTK) or OpenAI models for language processing. Jarvis can execute tasks like opening applications, sending emails, fetching information, and controlling IoT devices. With OpenCV and Mediapipe, it enables real-time gesture control and facial recognition. This versatile assistant bridges voice, visual, and text interfaces, offering an engaging AI-powered user experience.

Hand mouse

Description:-

A Python project that uses computer vision to track hand landmarks and control the mouse cursor. By leveraging Mediapipe for hand detection and OpenCV for video input, it maps specific finger movements to cursor actions, enabling hands-free control. Ideal for accessibility solutions and innovative human-computer interaction experiments.
